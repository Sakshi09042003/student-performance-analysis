
# Student Performance Analysis 📊

## Project Overview
This project analyzes student performance using Python and Pandas.
The goal is to understand score distribution and overall performance trends.

## Tools & Technologies
- Python
- Pandas
- Matplotlib

## Dataset
The dataset contains marks of students in:
- Math
- Science
- English

## Key Analysis
- Data cleaning
- Statistical summary
- Average score calculation
- Score distribution visualization

## How to Run
1. Clone the repository
2. Install requirements  
   `pip install -r requirements.txt`
3. Run the analysis script  
   `python src/analysis.py`

## Output
- Summary statistics
- Histogram of average student scores
